# iRIC_Grid_tester

2次元格子の各種スカラーのダミーデータを出力するデバッグ用ソルバーです。
* 格子点の実数値
* 格子点の整数値
* セルの実数値
* セルの整数値
* i方向境界麺の実数値
* i方向境界麺の整数値
* j方向境界麺の実数値
* j方向境界麺の整数値
* パーティクル
* パーティクルグループ
* パーティクル(画像)
